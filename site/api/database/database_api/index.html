    database.database_api ¶

    database_api.py

    Einheitliche Database API für Veritas RAG System Unterstützt verschiedene Database-Backends über eine gemeinsame Schnittstelle Mit integriertem Adaptive Batch Processing für optimale Performance

    Part of UDS3 (Unified Database Strategy v3) Author: Martin Krüger (ma.krueger@outlook.com) License: MIT with Government Partnership Commons Clause Repository: https://github.com/makr-code/VCC-UDS3
